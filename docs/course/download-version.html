Energy 对 CEF 版本支持 ​

  • Windows
    • Windows 10, 11 energy-latest
    • Windows 7, 8/8.1 and Windows Server 2012 CEF 109.1.18
    • Windows XP SP3 CEF 49.0.2623
  • MacOSX
    • 默认 : energy-latest
  • Linux
    • GTK3 默认 : energy-latest
  • Flash 版本
    • CEF 87.1.14
    • 取决于你使用的 liblcl 动态链接库
    • 例如: liblcl-89 只能在 CEF 87.1.14 使用
