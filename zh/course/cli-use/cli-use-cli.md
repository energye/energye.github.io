# CLI 自身管理命令

---

### energy cli

当前 CLI 版本，查看、更新

使用 `energy cli -h` 列出命令参数说明

```cmd
Usage:
  energy [OPTIONS] cli [cli-OPTIONS]

Help Options:
  /?                 Show this help message
  /h, /help          Show this help message

[cli command options]
      /v, /version   energy cli version
      /u, /update    energy cli update
```

### 功能说明

- CLI 版本查看
- CLI 更新
