# 组件库

组件库位于设计器左侧，提供所有可用的 UI 组件，按功能分类组织。组件库支持搜索过滤，方便快速查找。

## 搜索组件

组件库顶部提供 **搜索框**，输入关键词可快速过滤组件。搜索框支持中英文模糊匹配。

## 使用流程

1. 在组件库中找到目标组件（可通过搜索或手动浏览分类）
2. **单击** 组件名称，组件图标高亮表示已选中
3. 在设计画布上 **单击** 鼠标左键放置组件
4. 放置后自动切换回选择指针工具
