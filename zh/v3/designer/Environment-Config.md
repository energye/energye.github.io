# 环境配置

环境配置用于设置设计器运行所需的 Go 语言环境，通过 **设置 > 环境配置**（快捷键 `Ctrl+F10` / macOS `Cmd+F10`）打开。

## Go Root 路径

设置 Go 语言安装根目录（GOROOT）。设计器使用此路径查找 Go 编译器和工具链。

| 配置项 | 说明 | 输入类型 |
|--------|------|----------|
| Go Root | Go SDK 安装根目录路径 | 可编辑下拉框 + 目录选择器 |

**使用方式：**
- 点击下拉框可选择之前配置过的 Go 安装路径
- 点击目录选择器按钮，浏览并选择 Go 安装目录
- 设计器会验证所选目录下是否存在 `bin/go`（Linux/macOS）或 `bin\go.exe`（Windows）

**保存后的效果：**
- 设置 `GOROOT` 环境变量为所选路径
- 将 `<Go Root>/bin` 添加到 `PATH` 环境变量
- 配置保存在设计器全局配置文件 `~/.energy/config.json` 中，对当前项目生效

## Go 版本检测

设计器会自动检测系统中已安装的 Go 版本，并显示检测状态。

**要求：**
- Go 1.20 或更高版本为必需
- 检测失败时需手动指定 Go Root 路径

## 环境变量

设计器会自动初始化以下 Go 相关环境变量，确保依赖下载正常工作：

| 环境变量 | 说明 |
|----------|------|
| `GOROOT` | Go 安装根目录 |
| `GOPROXY` | Go 模块代理地址 |
| `GOSUMDB` | Go 校验数据库 |
| `PATH` | 包含 Go 工具链路径 |

## 配置保存

环境配置保存在设计器的全局配置文件 `.energy` 中，对所有项目生效。修改后点击"保存"按钮即刻生效。
