# 快捷键参考

## 文件操作

| 快捷键                    | macOS               | 功能      |
|------------------------|---------------------|---------|
| `Ctrl+P`               | `Cmd+P`             | 新建项目    |
| `Ctrl+N`               | `Cmd+N`             | 新建窗体    |
| `Ctrl+O`               | `Cmd+O`             | 打开项目    |
| `Ctrl+S`               | `Cmd+S`             | 保存      |
| `Ctrl+Q`               | `Cmd+Q`             | 退出      |

## 编辑操作

| 快捷键         | macOS     | 功能     |
|-------------|-----------|--------|
| `Ctrl+Z`    | `Cmd+Z`   | 撤销     |
| `Ctrl+Y`    | `Cmd+Y`   | 重做     |
| `Ctrl+X`    | `Cmd+X`   | 剪切     |
| `Ctrl+C`    | `Cmd+C`   | 复制     |
| `Ctrl+V`    | `Cmd+V`   | 粘贴     |
| `Ctrl+A`    | `Cmd+A`   | 全选     |
| `Delete`    | `Delete`  | 删除     |

## 视图控制

| 快捷键                 | macOS              | 功能            |
|---------------------|--------------------|---------------|
| `Ctrl+Alt+C`        | `Cmd+Alt+C`        | 显示/隐藏组件库     |
| `Ctrl+Alt+P`        | `Cmd+Alt+P`        | 显示/隐藏项目管理器   |
| `Ctrl+Alt+O`        | `Cmd+Alt+O`        | 显示/隐藏对象检查器   |
| `Ctrl+Alt+L`        | `Cmd+Alt+L`        | 显示/隐藏日志控制台   |
| `Ctrl+Alt+S`        | `Cmd+Alt+S`        | 显示/隐藏状态栏     |

## 构建与运行

| 快捷键                    | macOS              | 功能        |
|------------------------|--------------------|-----------|
| `F9`                   | `F9`               | 运行预览 / 停止 |
| `Ctrl+F8`              | `Cmd+F8`           | 构建        |
| `Ctrl+Shift+F8`        | `Cmd+Shift+F8`     | 清理构建      |
| `Ctrl+Shift+F9`        | `Cmd+Shift+F9`     | 构建所有      |

## 设置

| 快捷键          | macOS       | 功能     |
|--------------|-------------|--------|
| `Ctrl+F9`    | `Cmd+F9`    | 构建选项   |
| `Ctrl+F10`   | `Cmd+F10`   | 环境配置   |
| `Ctrl+F11`   | `Cmd+F11`   | 应用配置   |
