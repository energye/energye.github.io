# 日志控制台

日志控制台位于设计器底部，用于显示设计器运行日志、构建输出、预览运行信息等。

## 界面说明

日志控制台采用语法高亮编辑器（SynEdit），支持：
- 只读模式 - 仅显示日志，不可直接编辑
- 时间戳 - 每条日志自动添加时间戳 `[HH:MM:SS.sss]`
- 语法高亮 - 按日志级别显示不同颜色
- 自动滚动 - 新日志添加后自动滚动到底部

## 日志级别

| 级别    | 颜色   | 说明                  |
|-------|------|---------------------|
| ERROR | 红色   | 错误信息，需要关注          |
| WARN  | 蓝色   | 警告信息，可能存在潜在问题      |
| INFO  | 绿色   | 普通信息，如构建开始/结束、运行状态 |
| DEBUG | 灰色   | 调试信息，开发调试用          |

## 右键菜单

在日志控制台中 **右键** 弹出上下文菜单：

| 操作   | 说明          |
|------|-------------|
| 复制   | 复制选中的日志文本   |
| 全选   | 选中所有日志文本     |
| 清空   | 清空所有日志内容     |

## 日志来源

日志控制台接收以下来源的输出：

- **设计器日志** - 设计器自身的运行日志
- **构建输出** - 项目编译过程的输出
- **运行输出** - 预览运行时的应用输出
- **事件系统** - 各模块通过事件系统发送的消息

## 日志限制

为防止内存占用过大，日志控制台自动限制最大行数为 500 行。超过限制时，最早的日志会被自动删除。
