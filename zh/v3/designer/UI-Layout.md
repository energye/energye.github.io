# 界面布局

ENERGY Designer 的主窗口采用经典的 IDE 布局风格简化设计。界面由以下区域组成：

```
┌─────────────────────────────────────────────────────────────────┐
│  菜单栏: 文件 | 编辑 | 视图 | 运行 | 设置 | 帮助                    │
├─────────────────────────────────────────────────────────────────┤
│  工具栏: [新建窗体] [打开] [保存] [撤销] [重做] [运行预览]           │
├────────┬──────────┬───────────────────────────┬────────────────┤
│        │          │                           │                │
│  组件库  │ 项目管理器 │      设计器画布             │  对象检查器      │
│        │          │   (窗体设计 + 代码编辑)       │  (属性 + 事件)   │
│        │          │                           │                │
│        │          │                           │                │
│        │          ├───────────────────────────┤                │
│        │          │        日志控制台            │                │
├────────┴──────────┴───────────────────────────┴────────────────┤
│  状态栏: [选中组件] [位置/尺寸信息]                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 各面板说明

### 组件库（左侧）

显示所有可用的 UI 组件，按分类组织。支持搜索过滤。详见 [组件库](Component-Library.md)。

### 项目管理器（左中）

以树形结构展示项目的窗体组件树和源码文件。详见 [项目管理器](Project-Manager.md)。

### 设计器画布（中央）

核心设计区域，包含：
- **设计窗体标签页** - 多标签页并行设计多个窗体
- **窗体设计页** - 可视化拖拽设计 UI
- **代码编辑页** - 查看和编辑窗体对应的 Go 代码和 UI 文件

### 对象检查器（右侧）

查看和编辑选中组件的属性和事件。详见 [对象检查器](Object-Inspector.md)。

### 日志控制台（底部）

显示设计器运行日志、构建输出等信息。详见 [日志控制台](Log-Console.md)。

### 状态栏（最底部）

显示当前选中组件名称、组件位置和尺寸信息。

## 面板显示控制

所有面板均可通过 **视图** 菜单独立控制显示/隐藏，详见 [视图管理](View-Management.md)。

## 面板大小调整

各面板之间有分隔条（Splitter），拖拽分隔条可调整面板宽度。面板支持最小宽度限制，防止被完全拖没。
